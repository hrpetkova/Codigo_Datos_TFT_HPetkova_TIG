Observaciones:
Antes de cargar ningún dato, es esencial estar familiarizado con las características, condiciones y limitaciónes de la carga de los datos.
Esto se debe a que no todo Catálogo Sísmico lleva todos los datos que nos son necesarios, ni en todas las páginas donde podemos descargar
datos de zonas sísmicas podemos tener todo lo necesario.
En caso de querer probar con datos fuera de los pre-establecidos, los campos que no tenemos deben llevar valor tipo númerico o texto, dentro de los parámetros
de lo pre-establecido.
